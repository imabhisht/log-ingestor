How to Run:

 - First run all the Docker Contianers:
 `sudo sh start.sh`
 - Run Python Query App
 `python3 query/app.py`

 - Apache Kafka
 - Elastic Search (Also added Full Text Search Capabilities)
 - Docker 
 - HTML/JS for UI (Still under development, but API's are working)
